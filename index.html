Emoticore Command List

Statistics

++uses [emote]

Get an emote's usage count. You may mention an emote or use its name.

++stats (user)

Get a user's stats. Defaults to yourself, ping someone to specify a user.

++leaderboard (most-used|most-reacts-sent|most-reacts-received) (page)

Aliases: lb

Shows leaderboards for one of the categories, or a list of categories if none is specified.

Utility

++added [emote]

Shows when an emote was added. You may mention an emote or use its name.

++search [name]

Searches for emotes that match the search query. Type -g after the query to limit results to this server.

++em [emote]

Get the link to an emote. You may mention an emote or use its name.

Info

++help

Get a list of commands.

++info

Get info about the bot.

Logging

++log-channel [channel]

Set the logging channel, or disable logging if no channel is provided. To specify a channel, mention it. Requires permission to manage messages.

Misc.

++random

Shows usage stats for a random emote.

++user

Shows stats for a random user.
